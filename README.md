# 📚 LiterAlura

Aplicação Java desenvolvida com **Spring Boot** que consome a API pública do **Gutendex** para buscar livros, armazená-los em banco de dados e realizar consultas sobre livros e autores.

O projeto foi desenvolvido como parte do desafio **LiterAlura da Alura**, com foco em práticas de **consumo de API, persistência com JPA e manipulação de dados**.

---

# 🚀 Funcionalidades

A aplicação funciona via **menu no terminal** e permite:

- 🔎 Buscar livros pelo título na API
- 📖 Listar livros registrados no banco de dados
- 👨‍💻 Listar autores registrados
- 📅 Listar autores vivos em um determinado ano
- 🌎 Listar livros por idioma

---
# 🛠️ Tecnologias utilizadas

- **Java 17+**
- **Spring Boot**
- **Spring Data JPA**
- **Hibernate**
- **Maven**
- **PostgreSQL**
- **Jackson**
- **API Gutendex**
